# Cinderella Audition LP

`Cinderella entertainment` のオーディションLP用リポジトリです。

現在の主な公開対象ファイル:
- `index.html`
- `cinderella_audition_lp_html.html`

主要アセット:
- `assets/`

## リポジトリ概要

- 目的: GitHub上で確認・公開できるLPを制作する
- 対象: Cinderella entertainment メンズアイドル新メンバー募集LP
- 確認方法: ローカル表示 / GitHub / GitHub Pages を前提とする

## 制作ルール

- LPは最終的にGitHub上で確認・公開できる状態を目指す
- `Design.md` がある場合は、ほぼ正として扱う
- LP全体像の画像を先に作り、その後に構成を詰める
- 追加ヒアリングは不足情報がある場合のみ行う
- レスポンシブ対応は必須
- アニメーションは案件に応じて許容する

## 実装方針

- Next.js を第一候補とする
- フロントページが確認しやすい構成を優先する
- ダミー画像と本番画像の差し替えを容易にする
- セクション単位で見通しのよい実装を行う
- GitHub上でレビューしやすい構造を意識する
- 過剰な複雑化は避ける

補足:
- 本リポジトリは現状、単体HTML構成で実装されている
- 将来的に React / Next.js へ移行する場合も、セクション単位の見通しと差し替えやすさを維持する

## GitHub運用ルール

- GitHubでフロントのデザイン確認を行う
- 表示確認しやすい構成にする
- 公開URLで見せられる状態を目指す
- 差し替え対象画像は明示する
- ダミー画像利用箇所と本番差し替え箇所を整理する

## 現在の運用メモ

- GitHub Pages 公開を想定し、`index.html` を配置済み
- `.nojekyll` を配置済み
- 画像は `assets/` に集約して管理する
- 差し替え時は、既存参照パスを極力維持する

## ローカル確認

単体HTMLを直接開くか、必要に応じてローカルサーバーで確認します。

例:

```bash
python3 -m http.server 4173
```
